📊 Loan Portfolio Dashboard (Google Sheets)
Project Description

Designed and developed an interactive loan portfolio dashboard using Google Sheets to analyze lending performance, borrower behavior, and portfolio risk. The solution transformed raw loan data into actionable insights through automated data cleaning, dynamic KPIs, and visual reporting.

Built a structured data pipeline by importing and cleaning datasets using advanced spreadsheet functions, enabling accurate tracking of loan amounts, repayment status, and borrower distribution. Developed key performance indicators including total loan disbursed, total borrowers, default rate, and repayment totals to support decision-making.

Created interactive visualizations such as pie charts (loan status distribution), bar charts (loan amount by region), and line charts (monthly loan trends), allowing stakeholders to quickly identify patterns and high-risk segments. Integrated slicers and filters to enhance usability and enable real-time data exploration.

Key Achievements
Automated data cleaning using QUERY functions, reducing manual processing time
Built KPI metrics to monitor portfolio performance and credit risk
Developed an interactive dashboard improving data visibility and reporting efficiency
Enabled identification of default trends and high-risk borrower segments


Tools & Skills Used
Tools: Google Sheets
Skills: Data Cleaning, Data Analysis, Data Visualization, Dashboard Design, KPI Development
